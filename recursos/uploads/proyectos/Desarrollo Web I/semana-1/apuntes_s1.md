## 📌 Conceptos clave

### El protocolo HTTP
- **HTTP** (HyperText Transfer Protocol): protocolo de comunicación cliente-servidor
- **HTTPS**: versión segura con cifrado SSL/TLS
- Métodos: `GET`, `POST`, `PUT`, `DELETE`

### El navegador como cliente
El navegador interpreta tres lenguajes:
| Lenguaje | Rol |
|----------|-----|
| HTML | Estructura del contenido |
| CSS  | Presentación visual |
| JS   | Comportamiento interactivo |

### Editores recomendados
- ✅ **VS Code** — gratuito, extensible, el más usado
- ✅ **WebStorm** — potente pero de pago
- ✅ **Sublime Text** — ligero y rápido

## 💡 Tips de la clase
1. Siempre usar `<!DOCTYPE html>` al inicio
2. Las etiquetas se **abren y cierran**: `<p>texto</p>`
3. Las etiquetas vacías: `<br>`, `<img>`, `<input>`
