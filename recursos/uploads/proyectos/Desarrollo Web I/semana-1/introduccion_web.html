🌐 Introducción al Desarrollo Web

¿Qué es el desarrollo web?

El desarrollo web es el proceso de crear y mantener sitios y aplicaciones accesibles a través de internet. Se divide en:

  • Frontend: lo que el usuario ve (HTML, CSS, JavaScript)
  • Backend: la lógica del servidor (Node.js, Python, Java)
  • Full Stack: combinación de ambos
💡 Dato clave: HTML define la estructura, CSS el estilo y JavaScript el comportamiento.

Estructura básica HTML

<!DOCTYPE html>
<html lang="es">
  <head>
    <meta charset="UTF-8">
    <title>Mi primera página</title>
  </head>
  <body>
    <h1>¡Hola mundo!</h1>
  </body>
</html>

Herramientas del desarrollador

  • Visual Studio Code (editor)
  • Chrome DevTools (depuración)
  • Git (control de versiones)
  • Node.js (entorno de ejecución)
