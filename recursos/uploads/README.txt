📁 ESTRUCTURA DE ARCHIVOS – PORTAFOLIO JAZMINE JZ
=================================================

Esta carpeta contiene todos los archivos del portafolio académico organizados
por curso y semana. El sistema los registra y referencia automáticamente.

ESTRUCTURA:
───────────
uploads/
 ├── cursos/
 │    ├── desarrollo-web-1/
 │    │    ├── semana-1/     ← Coloca aquí los archivos de la Semana 1
 │    │    ├── semana-2/
 │    │    └── ... (hasta semana-16)
 │    ├── calculo-diferencial/
 │    │    └── semana-1/ ... semana-16/
 │    ├── programacion-1/
 │    │    └── semana-1/ ... semana-16/
 │    ├── base-de-datos-1/
 │    │    └── semana-1/ ... semana-16/
 │    └── matematica-discreta/
 │         └── semana-1/ ... semana-16/
 ├── perfil/
 │    ├── foto/     ← Foto de perfil
 │    └── cv/       ← Currículum en PDF
 ├── certificados/ ← Certificados y diplomas
 ├── proyectos/    ← Capturas o archivos de proyectos
 └── usuarios/     ← Datos de usuarios (si aplica)

TIPOS DE ARCHIVOS SOPORTADOS:
──────────────────────────────
 · Documentos: PDF, DOCX, PPTX, XLSX
 · Código:     HTML, CSS, JS, PY, SQL, JSON, TS
 · Texto:      TXT, MD
 · Imágenes:   PNG, JPG, JPEG, GIF, SVG, WEBP
 · Comprimidos: ZIP, RAR

INSTRUCCIONES:
──────────────
1. Copia el archivo físico en la carpeta correspondiente
   (ej: recursos/uploads/cursos/desarrollo-web-1/semana-3/practica.pdf)

2. Ve al Panel Admin → Archivos → Subir archivo

3. Selecciona el archivo, elige el curso y la semana correspondiente

4. Haz clic en "Registrar archivo"

El sistema guardará la referencia en localStorage y el archivo
quedará accesible para todos los visitantes del portafolio.
